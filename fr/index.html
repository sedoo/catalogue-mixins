# Démarrage rapide

Installer la dépendance

npm i @sedoo/catalogue-mixins

Dans main.js

import CatalogueMixins from "@sedoo/catalogue-mixins";
Vue.use(CatalogueMixins);
Attention Les mixins seront disponibles dans tous les composants du projet.
