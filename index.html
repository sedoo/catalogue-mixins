# Quick start guide

Install dependency

npm i @sedoo/catalogue-mixins

In main.js

import CatalogueMixins from "@sedoo/catalogue-mixins";
Vue.use(CatalogueMixins);
Warning The mixins will be available globally for all the components of the project.
